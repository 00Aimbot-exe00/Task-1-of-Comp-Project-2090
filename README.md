# Comp-Project-2090
README: Personal Finance Dashboard

Quick Start:
------------
Download through the terminal 
 python3 -m venv venv
 python -m pip install --upgrade pip
 python -m pip install PyQt6 matplotlib
 
Run the app
 python finance_app.py

Overview:
---------
This project provides a personal finance dashboard interface for tracking
income, expenses, assets, and net worth. It is designed to help users
visualize their financial health with charts, graphs, and categorized data.

Contents:
---------
- Account Information:
  * Available Balance
  * Total Net Worth

- Income Sources:
  * E-commerce
  * Google AdSense
  * My Shop
  * Salary

- Spending Categories:
  * Housing
  * Personal
  * Transportation
  * Pets (Routine Vet, Food, Treats, Boarding)

- Assets:
  * Gold
  * Stock
  * Warehouse
  * Land

- Progress Tracking:
  * Monthly income goals
  * Expense vs. income comparison
 
Installation: 
-------------
Run the Python script to launch a Ptqt6 GUI that displays the dashboard.
The interface is organized into frames for income, expenses, assets, and alerts.

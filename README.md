# Comp-Project-2090
README: Personal Finance Dashboard

Overview:
---------
This project provides a personal finance dashboard interface for tracking
income, expenses, assets, and net worth. It is designed to help users
visualize their financial health with charts, graphs, and categorized data.

Contents:
---------
- Account Information:
  * Available Balance
  * Total Net Worth

- Income Sources:
  * E-commerce
  * Google AdSense
  * My Shop
  * Salary

- Spending Categories:
  * Housing
  * Personal
  * Transportation
  * Pets (Routine Vet, Food, Treats, Boarding)

- Assets:
  * Gold
  * Stock
  * Warehouse
  * Land

- Notifications:
  * Alerts for overdue bills

- Progress Tracking:
  * Monthly income goals
  * Expense vs. income comparison
 
Installation: 
-------------
Run the Python script to launch a Tkinter-based UI that displays the dashboard.
The interface is organized into frames for income, expenses, assets, and alerts.

Quick Start:
------------
Download through the terminal 
  python3 --version
Open 
python3 dashboard.py
